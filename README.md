# ⚽ Football Global Transfers Analytics (2010–2026)

## 📌 Project Overview

This project analyzes global football transfers between 2010 and 2026 using SQL Server and Power BI.

The goal is to transform raw transfer data into meaningful business insights through data modeling, ETL processes, SQL analysis, and interactive dashboards.

---

## 🎯 Objectives

- Analyze transfer spending over time.
- Identify clubs with the highest transfer expenditures.
- Discover the most expensive player transfers.
- Compare leagues and clubs.
- Build an interactive Power BI dashboard.
- Apply SQL and data modeling best practices.

---
